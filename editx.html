GitHub Setup

Connect your repository

The dashboard commits directly to your GitHub repo. Your token is stored only in your browser.

1Go to github.com/settings/tokens/new, set expiry No expiration, Fine-grained token, select your repo, grant Contents: Read & Write.
2Paste token and repo details below and click Save.
3Every publish commits directly to GitHub — live within seconds.
