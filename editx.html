GitHub Setup

Connect your repository to publish changes live

The dashboard commits directly to your GitHub repo. Your token is stored only in your browser.

1Go to github.com/settings/tokens/new, set expiry to No expiration, choose Fine-grained token, select your repo, grant Contents: Read & Write.
2Paste your token and repository details below and click Save.
3Every time you publish an item or save a page, it commits directly to GitHub — your site updates within seconds.
